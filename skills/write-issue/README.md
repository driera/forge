# write-issue

Creates a GitHub issue in user-story format. You provide the what and why; the skill drafts the full issue, lets you refine it, then creates it on GitHub.

## When to use

Say "write an issue", "create an issue", "let's add an issue for X", or "open a ticket for".

## What it produces

A GitHub issue with:
- User story title: "As a [user], I want [action] so that [outcome]"
- Context section
- Acceptance criteria (specific, testable, includes a11y and tests)
- Technical notes
- Definition of done

## Closing nudge

After creation: "Issue #NNN created. When you're ready to start, say `explore issue NNN`."
